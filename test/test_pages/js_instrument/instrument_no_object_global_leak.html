Test that the legacy JavaScript instrument does not assign its getPropertyDescriptor / getPropertyNames helpers as own properties of the page-world Object global. Such assignments are a config-independent fingerprinting tell (Krumnow, Jonker & Karsch, arXiv:2205.08890).

The browser instruments window.fetch (configured python side) so the instrument runs on this page. Native Object carries neither helper, so a clean instrument leaves Object.getOwnPropertyNames(Object) free of both names. We read whether either name is an own property of Object and encode the booleans into the arguments of an instrumented fetch call so they land in the javascript table for the test to assert on.
