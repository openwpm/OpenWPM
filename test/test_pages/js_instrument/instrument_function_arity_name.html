Test that an instrumented function wrapper reports the same .length (arity) and .name as the native function it wraps, so the wrapper is not trivially fingerprintable.

The browser instruments window.fetch (the subject under test) and window.name (a writable side channel), both configured python side. Native fetch has length === 1 and name === "fetch". We read the wrapper's observable .length and .name WITHOUT calling fetch, and exfiltrate them as a JSON object through window.name so the instrument records the structured values for the test to assert on.
