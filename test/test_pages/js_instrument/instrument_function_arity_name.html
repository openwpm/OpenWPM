Test that an instrumented function wrapper reports the same .length (arity) and .name as the native function it wraps, so the wrapper is not trivially fingerprintable.

The browser instruments window.fetch (configured python side). Native fetch has length === 1 and name === "fetch". We read the wrapper's observable .length and .name and encode them into the arguments of an instrumented fetch call so the values land in the javascript table for the test to assert on.
